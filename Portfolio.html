IEEE-Sensors-PI

Programming Team - Spring Project 2022 - SensorsPi IEEE Student Branch University of Western Macedonia, Kastoria

This project is based on receiving information from a temperature/humidity sensor connected to a Raspberry Pi and displaying them with the help of a python made GUI. The current project is able to track the temperature and humidity changes of the last 5 hours, representing the changes with the help of 2 diagrams.
